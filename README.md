# :necktie:  Smartcloset (내손안의 옷장)

##  :bulb: Smartcloset 이란
### "IoT 관점"의 기능 추가로 수납기능 뿐인 기존옷장에서 
### 청정관리, 스타일링 , 세탁알림 등 다양한 사용자 편의를 위해 서비스를 제공한다.
### 이외의 사용자의 스마트폰을 이용하여 날씨정보를 확인하고 기능제어와 추천의류 조회 등의 서비스를 제공한다.
